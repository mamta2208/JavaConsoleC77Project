Visual Studio Code

Visual Studio Code is a source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git

What is Console ?

The Console panel. Web developers log messages for 2 general reasons: Making sure that code is executing in the right order. Inspecting the values of variables at a certain moment in time.

How to view console

Right Click on the screen. Select the last option 'Inspect'. In the window opened, select 'Console'. You can view the values there.
